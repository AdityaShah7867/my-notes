# **App Name**: FocusFlow

## Core Features:

- Note Taking: Basic text editing with formatting options (bold, italics, underline).
- Local Storage Autosave: Automatically save notes to local storage, with configurable intervals.
- To-Do List: Create and manage to-do lists with task completion status and drag and drop functionality.
- Categorization: Organize notes with folders or tags.
- Search Functionality: Search through saved notes.

## Style Guidelines:

- Primary color: Calm Blue (#64B5F6) to promote focus.
- Background color: Light gray (#F0F4F8) for comfortable readability.
- Accent color: Soft Green (#81C784) for task completion and highlights.
- Body and headline font: 'PT Sans' sans-serif, combining modernity and warmth.
- Simple, minimalist icons for tasks and folders.
- Clean and intuitive layout with a sidebar for navigation and main content area for notes and to-do lists.
- Subtle transitions for opening notes and task completion.
